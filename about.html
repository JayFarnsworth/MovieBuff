Film Buff


This is a project completed for my Galvanize P2. The requirements were to make a web app that renders API data. It was far, far to ambitious for the project, and as such I have to move on before it is complete. It will continuously be added to and altered as my time allows moving forward.


What works:
Actor and Director Search will accurately render results, and can filter based on the Rotten Tomatoes minimum score, if given. Results are displayed in order of film popularity as determined by TMDB.

Profile actions work. You can create a profile, and add favorites to it. The fetch favorites button will grab the plain text from your profile.


What's coming:
I'm planning on introducing a much more robust set of results filters and sorts. This includes filtering by any of my three given review scores. Results will be able to be sorted by year, score, box office, etc.

I'm also planning on adding a filmography visualization feature that will graph an actor or director's films based on a number of different changeable criteria. The idea is to be able to chart a filmography based on year and quality.

Most importantly, I will be creating a more robust set of profile features. Upon adding a favorite, an API search query will be made, which will confirm the actual movie, actor, or director queried. The data retreived from the search will be added to your profile on Filmbuff's server. This will allow for a fetch of your user favorites to retreive full sets of data from each item instead of the simple text that was initially inputted. Links will be added to each movie, actor, and director to fetch their data with a click. Upon loading up a favorite director or actor in your profile, each film will have a "seen" button or a "favorite". This way you can track which movies from a director's filmography you haven't seen, and ideally will quickly match you with the high quality films from the artist you have not seen.

Finally I'm going to build out an algoritm that can reccomend movies to you based on your favorites. There are a number of ways that I can go about doing this. I'm going to place the most emphasis on the intersection of your favorite people and quality films.
